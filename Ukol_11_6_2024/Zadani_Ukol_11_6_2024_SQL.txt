Zadání EN:
Module Introduction to the Database Theory
Part 1
Task 0
Install MySQL Community Server.
Task 1
Create a database named Birds. The location is up to you.
Task 2
Rename the database from the first task. New name for the database - Cats.
Task 3
Delete the Cats database.
Task 4
Create a one-table Vegetables and Fruit database. You need to store the following information:
Name;
Type (vegetable or fruit);
Colour;
Calorie content;
Short description.
Task 5
Create the following queries for the table containing information about vegetables and fruit from the previous task:
Displaying of all data from the table about vegetables and fruit.
Displaying of all vegetables.
Displaying of all fruit.
Displaying of all the names of vegetables and fruit.
Displaying of all colors. Colors should be unique.
Displaying fruit of a specific color.
Displaying vegetables of a particular color.


Zadání CZ:
Část 1
Úkol 0
Nainstalujte komunitní server MySQL.
Úkol 1
Vytvořte databázi s názvem Birds. Umístění je na vás.
Úkol 2
Přejmenujte databázi z první úlohy. Nový název databáze - Kočky.
Úkol 3
Smažte databázi Cats.
Úkol 4
Vytvořte databázi zeleniny a ovoce s jednou tabulkou. Musíte uložit následující informace:
Název;
Druh (zelenina nebo ovoce);
Barva;
Obsah kalorií;
Stručný popis.
Úkol 5
Vytvořte následující dotazy pro tabulku obsahující informace o zelenině a ovoci z předchozí úlohy:
Zobrazení všech údajů z tabulky o zelenině a ovoci.
Vystavování veškeré zeleniny.
Vystavování veškerého ovoce.
Zobrazení všech názvů zeleniny a ovoce.
Zobrazení všech barev. Barvy by měly být jedinečné.
Zobrazení ovoce určité barvy.
Zobrazení zeleniny určité barvy.
